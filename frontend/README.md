# Frontend (Next.js)

This is the frontend for the Study Sprint AI technical assessment.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create local environment file:

```bash
cp .env.example .env.local
```

3. Start development server:

```bash
npm run dev
```

4. Open the app:

- http://localhost:3000

## Environment Variable

- `NEXT_PUBLIC_API_BASE_URL` (default in `.env.example` is `http://localhost:8000`)
